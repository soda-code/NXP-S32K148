◆ BSP_CAN0_Init()

void BSP_CAN0_Init ( void )

初始化CAN0控制器

该函数完成CAN0控制器的完整初始化流程，包括：

  • 初始化CAN控制器实例
  • 配置标准帧和扩展帧的接收/发送邮箱
  • 设置接收过滤器
  • 安装事件回调函数
  • 设置中断优先级
  • 启动接收过程
注解
该函数不返回任何值，所有配置均为硬件直接操作
警告
调用前需确保CAN0相关硬件已上电且时钟稳定
注意
该函数会覆盖所有CAN0相关配置，调用后需重新配置过滤器

◆ BSP_CAN0_Send_EXT()

uint8_t BSP_CAN0_Send_EXT ( uint32_t id,
uint8_t data[64],
uint8_t length )

通过CAN0总线发送扩展帧格式数据

参数
idCAN消息ID（扩展帧格式）
data待发送的数据缓冲区（最大64字节）
length实际数据长度（字节数）
返回
uint8_t 发送状态码：
  • 0：发送成功
  • 1：发送失败
  • 2：中止传输失败
注解
函数内部会尝试500次轮询发送状态，超时后将尝试中止当前传输并重发
警告
数据长度参数必须小于等于64字节，否则会导致内存越界

◆ BSP_CAN0_Send_STD()

void BSP_CAN0_Send_STD ( uint16_t id,
uint8_t data[8] )

通过CAN0总线发送标准帧数据

参数
id要发送的CAN消息ID（标准帧格式）
data要发送的8字节数据数组
注解
此函数会尝试发送CAN消息，如果发送邮箱繁忙，会等待最多500次尝试：
  1. 正常情况：等待发送邮箱空闲后发送
  2. 超时情况：强制中止当前传输并重新尝试发送
警告
调用此函数前必须确保CAN0已正确初始化
