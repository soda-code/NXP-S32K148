◆ BSP_CAN1_Init()

void BSP_CAN1_Init ( void )

初始化CAN1控制器

完成CAN1控制器的完整硬件配置，包括：

  • 控制器初始化
  • 标准帧和扩展帧的收发邮箱配置
  • 接收过滤器设置（全通模式）
  • 中断回调函数注册
  • 中断优先级设置
  • 启动接收操作
注解
该函数应在系统初始化阶段调用，且只调用一次
使用前需确保CAN1时钟已使能
中断优先级设置为0x02（较高优先级）

◆ BSP_CAN1_Send_EXT()

uint8_t BSP_CAN1_Send_EXT ( uint32_t id,
uint8_t data[64],
uint8_t length )

通过CAN1发送扩展帧数据（支持CAN FD）

参数
id29位扩展帧标识符
data待发送的数据缓冲区（最大64字节）
length数据长度（1-64字节）
返回
uint8_t 发送状态码：
  • 0: 发送成功
  • 1: 发送失败
  • 2: 中止传输失败
注解
该函数采用阻塞式发送，最大重试500次后强制中止
警告
数据长度超过64字节将导致内存越界

参数
