◆ BSP_UART0_Init()

void BSP_UART0_Init ( void )

初始化UART0外设

配置UART0硬件接口，包括：

  • 初始化UART控制器
  • 设置中断优先级
  • 启动首次接收操作
注解
该函数必须在其他UART0操作前调用
警告
中断优先级设置为0x02（较高优先级）
