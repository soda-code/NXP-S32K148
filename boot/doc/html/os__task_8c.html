◆ OS_Init()

void OS_Init ( void )

系统初始化函数，负责硬件和RTOS基础环境初始化

该函数执行以下关键初始化操作：

  1. 系统时钟初始化（48MHz配置）
  2. MCU电源模式设置（HSRUN高速运行模式）
  3. 板级引脚初始化
  4. 创建系统初始化任务（T_Init）
注解
此函数应在main()函数中首先调用，且只调用一次，通过PEX_RTOS_INIT宏定义控制是否调用
警告
调用前需确保已正确配置g_clockManConfigsArr等全局配置数组

[in]void无输入参数
返回
void 无返回值
