◆ BSP_CAN2_Init()

void BSP_CAN2_Init ( void )

初始化CAN2控制器

完成CAN2控制器的完整硬件初始化配置，包括：

  • 控制器基础配置
  • 标准帧和扩展帧的收发邮箱配置
  • 接收过滤器设置（全通模式）
  • 中断回调函数注册
  • 中断优先级设置
  • 启动接收操作
注解
该函数应在系统初始化阶段调用，且只调用一次
使用前需确保相关缓冲区配置已完成（Rx_buffCfg_STD/EXT, Tx_buffCfg_STD/EXT）
中断优先级设置为0x02（较高优先级）
