◆ MCU_Freq_Print()

void MCU_Freq_Print ( void )

打印MCU核心频率信息

该函数获取当前MCU的核心时钟频率，并通过UART1接口以MHz为单位输出频率信息。 输出格式为："[MCU INFO] Core Frequency: X MHz\r\n"

◆ OS_Task_Diagnosis()

void OS_Task_Diagnosis ( void * pvPara)

诊断任务主函数，周期性执行系统健康监测

该任务以固定周期运行系统健康检查，包括但不限于：

  • 系统资源监控
  • 关键组件状态检查
  • 故障诊断处理
参数
pvParaFreeRTOS任务参数（未使用）
注解
任务周期由DIAGNOSIS_PERIOD宏定义控制
使用vTaskDelayUntil确保精确周期调度
