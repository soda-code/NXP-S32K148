void OS_Task_Body ( void * pvPara)

车身控制任务主函数

该任务负责处理车身相关控制逻辑，包括LED状态控制，并定期发送任务存活信号。 任务以固定10ms周期运行，确保实时性。

参数
pvPara任务参数指针（未使用）
注解
任务通过设置事件标志位BODY_TASK_ALIVE_BIT来指示任务正常运行状态

上报任务存活状态
