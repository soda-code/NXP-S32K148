◆ BSP_UART0_Init()

void BSP_UART0_Init ( void )

初始化UART0外设

配置UART0硬件接口，包括：

  • 初始化UART控制器
  • 设置中断优先级
  • 启动首次接收操作
注解
该函数必须在其他UART0操作前调用
警告
中断优先级设置为0x02（较高优先级）

◆ UART0_Rx_Callback()

void UART0_Rx_Callback ( void * driverState,
uart_event_t event,
void * userData )

UART0接收中断回调函数

处理UART0接收中断事件，将接收到的数据通过消息队列发送给上层任务处理。 支持单字节接收模式和缓冲区管理，遇到换行符或缓冲区满时停止接收。

参数
driverState驱动状态指针（未使用）
event触发回调的事件类型
userData用户数据指针（未使用）
注解
此函数在中断上下文中执行，必须保持简短高效
使用FreeRTOS的FromISR版本队列API确保中断安全
警告
缓冲区大小由UART0_BUF_SIZE宏定义，修改需同步调整相关逻辑
