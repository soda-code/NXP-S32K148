初始化\+CAN2控制器 

完成\+CAN2控制器的完整硬件初始化配置，包括：
\begin{DoxyItemize}
\item 控制器基础配置
\item 标准帧和扩展帧的收发邮箱配置
\item 接收过滤器设置（全通模式）
\item 中断回调函数注册
\item 中断优先级设置
\item 启动接收操作
\end{DoxyItemize}

\begin{DoxyNote}{注解}
该函数应在系统初始化阶段调用，且只调用一次 

使用前需确保相关缓冲区配置已完成（\+Rx\+\_\+buff\+Cfg\+\_\+\+STD/\+EXT, Tx\+\_\+buff\+Cfg\+\_\+\+STD/\+EXT） 

中断优先级设置为0x02（较高优先级） 
\end{DoxyNote}
