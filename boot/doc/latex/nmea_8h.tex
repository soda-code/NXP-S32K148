\hline

\doxylink{struct_g_p_s}{GPS} 包含所有解析后\+GPS数据的结构体

\begin{DoxyNote}{注解}
根据编译时定义的宏(ENABLE\+\_\+\+GGA/\+ENABLE\+\_\+\+GLL等)决定解析哪些\+NMEA数据段 未启用的数据段将返回默认值 支持解析的\+NMEA数据段包括：
\begin{DoxyItemize}
\item \doxylink{struct_g_g_a}{GGA}(全球定位系统固定数据)
\item \doxylink{struct_g_l_l}{GLL}(地理定位信息)
\item \doxylink{struct_g_s_a}{GSA}(当前卫星信息)
\item \doxylink{struct_r_m_c}{RMC}(推荐最小特定\+GPS/\+TRANSIT数据)
\item \doxylink{struct_v_t_g}{VTG}(地面速度信息)
\item GSV(可见卫星信息)
\item UTC(时间信息) 
\end{DoxyItemize}
\end{DoxyNote}

void gps\+\_\+global\+\_\+init (\begin{DoxyParamCaption}\item[{void}]{}{}\end{DoxyParamCaption})}



初始化全局\+GPS数据结构 

该函数用于初始化全局\+GPS数据结构体gps\+\_\+all，根据编译时定义的宏(ENABLE\+\_\+\+GGA/\+ENABLE\+\_\+\+RMC等) 来初始化对应的\+NMEA语句解析数据结构。每个启用的\+NMEA语句类型都会被初始化为默认值。

\begin{DoxyNote}{注解}
该函数应在系统启动时调用一次，用于初始化\+GPS解析模块的全局数据结构 

当前实现不使用动态内存分配，所有缓冲区均为静态分配
\end{DoxyNote}
支持的\+NMEA语句类型包括：
\begin{DoxyItemize}
\item \doxylink{struct_g_g_a}{GGA}\+: 全球定位系统定位数据
\item \doxylink{struct_g_l_l}{GLL}\+: 地理定位信息
\item \doxylink{struct_g_s_a}{GSA}\+: 当前卫星信息
\item \doxylink{struct_r_m_c}{RMC}\+: 推荐最小定位信息
\item \doxylink{struct_v_t_g}{VTG}\+: 地面速度信息
\item GSV\+: 可见卫星信息
\end{DoxyItemize}

每个语句类型的初始化值均为零或空字符串
