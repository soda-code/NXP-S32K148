初始化\+UART2外设 

配置\+UART2硬件参数并启动接收中断。该函数执行以下操作：
\begin{DoxyEnumerate}
\item 初始化\+UART2硬件寄存器
\item 设置\+UART2中断优先级
\item 启动首次接收操作
\end{DoxyEnumerate}

\begin{DoxyNote}{注解}
调用此函数前需确保uart2\+\_\+instance和uart2\+\_\+\+Config0已正确配置 
\end{DoxyNote}
\begin{DoxyWarning}{警告}
此函数会启用\+UART2中断，需确保中断处理函数已正确实现 
\end{DoxyWarning}
