系统初始化函数，负责硬件和\+RTOS基础环境初始化 

该函数执行以下关键初始化操作：
\begin{DoxyEnumerate}
\item 系统时钟初始化（48\+MHz配置）
\item MCU电源模式设置（\+HSRUN高速运行模式）
\item 板级引脚初始化
\item 创建系统初始化任务（\+T\+\_\+\+Init）
\end{DoxyEnumerate}

\begin{DoxyNote}{注解}
此函数应在main()函数中首先调用，且只调用一次，通过\+PEX\+\_\+\+RTOS\+\_\+\+INIT宏定义控制是否调用 
\end{DoxyNote}
\begin{DoxyWarning}{警告}
调用前需确保已正确配置g\+\_\+clock\+Man\+Configs\+Arr等全局配置数组
\end{DoxyWarning}
