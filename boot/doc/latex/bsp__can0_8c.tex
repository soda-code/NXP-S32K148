初始化\+CAN0控制器 

该函数完成\+CAN0控制器的完整初始化流程，包括：
\begin{DoxyItemize}
\item 初始化\+CAN控制器实例
\item 配置标准帧和扩展帧的接收/发送邮箱
\item 设置接收过滤器
\item 安装事件回调函数
\item 设置中断优先级
\item 启动接收过程
\end{DoxyItemize}

\begin{DoxyNote}{注解}
该函数不返回任何值，所有配置均为硬件直接操作 
\end{DoxyNote}
\begin{DoxyWarning}{警告}
调用前需确保\+CAN0相关硬件已上电且时钟稳定 
\end{DoxyWarning}
\begin{DoxyAttention}{注意}
该函数会覆盖所有\+CAN0相关配置，调用后需重新配置过滤器 
\end{DoxyAttention}
