初始化\+CAN1控制器 

完成\+CAN1控制器的完整硬件配置，包括：
\begin{DoxyItemize}
\item 控制器初始化
\item 标准帧和扩展帧的收发邮箱配置
\item 接收过滤器设置（全通模式）
\item 中断回调函数注册
\item 中断优先级设置
\item 启动接收操作
\end{DoxyItemize}

\begin{DoxyNote}{注解}
该函数应在系统初始化阶段调用，且只调用一次 

使用前需确保\+CAN1时钟已使能 

中断优先级设置为0x02（较高优先级） 
\end{DoxyNote}
